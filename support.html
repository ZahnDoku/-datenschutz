Häufige Fragen

Wo werden meine Daten gespeichert?

Alle Daten (Patientenfotos, Namen, Annotationen) werden ausschließlich lokal auf Ihrem iPhone oder iPad gespeichert. Es findet kein Upload auf externe Server statt.

Ich habe meine PIN vergessen — was tun?

Tippen Sie auf „PIN vergessen?“ auf dem Sperrbildschirm. Sie können Ihre PIN über Face ID oder Touch ID zurücksetzen.

Wie erstelle ich ein Backup?

Gehen Sie zu Einstellungen → Backup → Backup erstellen. Das Backup wird als ZIP-Datei im lokalen Dateisystem gespeichert und kann über die Teilen-Funktion exportiert werden.

Kann ich gelöschte Fotos wiederherstellen?

Ja, gelöschte Fotos befinden sich 7 Tage im Papierkorb und können dort wiederhergestellt werden.

Welche Geräte werden unterstützt?

ZahnDoku läuft auf iPhones und iPads mit iOS 17.6 oder neuer.
